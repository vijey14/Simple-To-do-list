# Simple To-Do List

A clean, modern to-do list web application built with HTML, CSS, and JavaScript. Perfect for managing daily tasks with a beautiful and user-friendly interface.

## Features

- ✅ **Add Tasks** - Quickly add new tasks with a simple input field
- ✅ **Mark Complete** - Check off tasks as you complete them (with strikethrough effect)
- ✅ **Delete Tasks** - Remove individual tasks with the delete button
- ✅ **Filter Tasks** - View All, Active, or Completed tasks
- ✅ **Clear Completed** - Remove all completed tasks at once
- ✅ **Local Storage** - Tasks automatically save to your browser's local storage and persist across sessions
- ✅ **Task Counter** - See how many tasks are active and total tasks at a glance
- ✅ **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- ✅ **Beautiful UI** - Modern gradient design with smooth animations and transitions
- ✅ **XSS Protection** - HTML escaping to prevent injection attacks

## How to Use

### Getting Started

1. Open `index.html` in your web browser
2. You're ready to start! No installation or dependencies needed

### Adding a Task

- Type your task in the input field
- Press **Enter** or click the **Add** button
- The task will appear at the top of your list

### Managing Tasks

- **Mark Complete** - Click the checkbox next to a task to mark it as complete
- **Delete** - Click the **Delete** button to remove a task
- **Filter** - Use the filter buttons to view:
  - **All** - All tasks (default)
  - **Active** - Only incomplete tasks
  - **Completed** - Only completed tasks

### Clearing Tasks

- Click **Clear Completed** to remove all finished tasks at once

## Data Storage

All your tasks are automatically saved to your browser's **Local Storage**. This means:
- ✅ Tasks persist when you close and reopen the browser
- ✅ Tasks are stored locally on your device
- ✅ Clearing browser cache will remove your tasks
- ✅ Each browser/device has its own separate task list
